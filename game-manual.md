# 🧩 俄罗斯方块 · Tetris

> 经典俄罗斯方块，浏览器即开即玩，桌面端与手机端均可畅享。

---

## 🎮 游戏简介

七种彩色方块从天而降，你要做的是旋转、移动它们，填满整行就能消掉得分。随着等级提升，方块下落越来越快，挑战你的反应极限！

## 📋 游戏规则

- **下落**：方块自动匀速下落，你需要将它放到合适的位置
- **消行**：填满一行即可消除，同时消除的行数越多，单次得分越高
- **计分**：
  | 同时消除行数 | 得分 |
  |:----------:|:---:|
  | 1 行 | 100 |
  | 2 行 | 300 |
  | 3 行 | 500 |
  | 4 行 | 800 |
- **等级**：每消除 10 行升 1 级，等级越高下落越快
- **Game Over**：方块堆到顶部放不下时游戏结束

## ⌨️ 键盘操作（桌面端）

| 按键 | 操作 |
|:---:|:----:|
| `←` `→` | 左右移动 |
| `↑` | 旋转 |
| `↓` | 软加速（按住持续下落） |
| `空格` | 硬降到底 |
| `P` | 暂停 / 继续 |
| `R` | 游戏结束后重新开始 |

## 💡 小贴士

1. **提前看下一块** — 右侧面板会预告下一个方块，提前规划位置
2. **留一列空隙** — 方块堆整齐，留出缝隙应对长条 I 型
3. **先旋转再硬降** — 不要硬降到底才发现方向不对
4. **活用软加速** — 按 `↓` 或下滑微调方块位置
5. **四行消最赚** — 一口气消 4 行得 800 分，比分 4 次单行多 400 分
6. **看准影子** — 方块下方的半透明影子就是落点，提前判断
7. **心态稳住** — 等级越高下落越快，冷静应对才是高分秘诀
